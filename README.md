## Cara Instalasi
Untuk menginstal JamTeng, cukup jalankan perintah berikut di terminal Anda:

```bash
dpkg -i jamTeng.deb
```

Setelah instalasi selesai, buat alias untuk memudahkan akses:

```bash
alias jamteng='/opt/jamTeng/jamTeng'
```
